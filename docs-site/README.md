# JobOps Docs Site

Docusaurus workspace for user-facing documentation.

## Development

From repository root:

```bash
npm run docs:dev
```

Local docs server runs on `http://localhost:3006`.

## Build

```bash
npm run docs:build
```

Build output:

- `docs-site/build`

## Versioning

Cut a docs version tied to a release tag:

```bash
npm run docs:version -- 1.0.0
```
